03
✍️

맞춤법 검사

Claude API 키 필요

📄

보고서 초안 생성

과제명·배경·이슈를 입력하면 대표이사 보고 형식의 초안을 즉시 생성합니다.

Claude API 키 필요
시작 방법: 프로젝트/리소스 관리는 API 키 없이 바로 사용 가능합니다. 회의록·보고서 AI 생성 기능을 사용하려면 우측 상단 API 키 설정에서 Claude API 키를 입력하세요. 키는 이 세션 메모리에만 보관되며 브라우저를 닫으면 자동 삭제됩니다.
